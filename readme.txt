== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Element Helperor'
3. Activate Element Helperor from your Plugins page.

=== Manually ===

1. Upload the `elh-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the Element Helperor plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
